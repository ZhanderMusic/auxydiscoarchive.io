akabillposters:

One of the things I like about SC (Pro accounts), is the ability to update/replace the source audio file without losing play counts, etc… Is that possible using something like Amuse?

AH! This is very interesting.

So every track you put on Spotify or other stores has an “ISRC”. You may know about this already, but here’s an overview: https://en.wikipedia.org/wiki/International_Standard_Recording_Code

Basically, if your track is distributed onto Spotify in a different fashion (e.g. a compilation album), and they have the same artist name, title, song length, and ISRC, they will count the plays as the same. HOWEVER, an ISRC is needed for each edition of a song. Usually this would be like Radio Edit, Extended Mix, or Alternate Version, but if you want to actually change the song, it has the same effect. So unless the second version is the same song length, you can use the same ISRC and keep the stats!
