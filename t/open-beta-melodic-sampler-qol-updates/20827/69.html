Auxy April 6, 2021, 3:09pm #80

New build (845) posted with the following tweaks:

  • Added a master gain slider in the project settings (pre master effects)
  • Reverted to using the previous limiter settings (i.e. old projects should sound like in the App Store version again, unless you change the gain)
  • Fixed a bug that would cause imported samples to show up as duplicates
  • Improved the sample pitch menu to help set the correct tuning for imported samples
  • Melodic samples are now included when sharing projects

Keep testing and let us know if you have any issues! If you get a crash and see the option to send more info to the developer, please do so so that we can investigate further.
