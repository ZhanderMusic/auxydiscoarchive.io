Iso August 8, 2017, 3:38pm #50

I’ve wanted this for a while and am curious to see if anyone else likes the idea… Parameter Oscillators!
An LFO with a selectable waveform than can be used to modulate instrument parameters.

Example:

  • pick an oscillator and set it to a waveform: square, sine, triangle, noise, semisine etc
  • set its timing signature similar to how you select the delay timing signature e.g. 1/4
  • route the oscillator to a parameter e.g. reverb, bitcrush etc

Its possible to get something similar working with a bit of work with the new automation feature, but I’d really love this to be a standalone feature. Of course UI space is at a premium, but if it were to go ahead, the logical space for it would be in the instrument settings popup where tone, delay, reverb, shape etc are housed… I think a noise waveform could be particularly interesting

Iso August 8, 2017, 10:53pm #57

I worded the feature/concept quite badly and probably should post a UI mockup at some point? Besides, modulation is probably a better term to use… Perhaps this scenario may explain better.

  • You have Bee bass synth selected and the Tone parameer set all the way to the right at 100%.
  • You want the LOWPASS parameter to transition from 0% to 100% (or 25% to 75%) and back again over a period of 1 bar.
  • Click on the hypothetical LFO/oscillator knob in the instrument parameters popup and
    • set its ‘destination’ to the Lowpass parameter
    • set its oscillation rate to 4 beats (imagine pressing the little asterisk symbol underneath DELAY)
    • set its minimum value to 0% and its maximum value to 100% (or any other percentage)
    • set its waveform type to sine, triangle, square etc

The LOWPASS value changes from 0% to 100% at the fixed rate. You can think of this almost as a variation of the transition feature, with the difference that it oscillates values over and back.

That may not make much sense but I hope you get the idea!
