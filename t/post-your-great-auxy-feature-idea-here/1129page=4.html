ImOnlySleeping August 9, 2017, 5:51pm #64

I think an LFO like many have said on here is a good idea. It allows you to program automation that is a bit too complicated to draw in with lines and dots.

It’s a time saver and when it comes to music every second counts.
A good idea can be lost in an instant!

I would also like to see a visual way to see if your autuomation track has any info.

Like I’ll make an automation clip with more than one parameter changed but if I want to go back and edit something, there’s no way of knowing which parameters have been affected aside from clicking all of them and seeing for yourself.

So basically just a dot or marker next to a parameter so we don’t have to click them all just to see which one we want to fix.

Pitching hi hats and the rest of the drums would be a great feature. Hi hats especially for trap/hip hop. Individual pitching of all the drums

I know the whole point of auxy is to be simple and contained but it’s such a great app I can’t help but want more from it.

It would also be nice to have a drumsynth style kit. Where you could shape and edit your kicks and snare with parameters like you would if you were to make them on a synth yourself.
It would require each element to have its own parameter window but I think most users would appreciate drums that don’t sound like everyone else’s.

Different selectable reverb types i.e.

Spring reverb
Plate reverb

Lastly, I think a sampler type instrument would be cool. Found sounds/foley/field recordings would add a whole new dimension to this app and introduce users to the joy of sampling lol

It would have

The options:
Pitching (not just with the piano roll
Attack
Release
Reverb
Delay
Hi pass
Low pass
Chorus

Editing menu would let you:
Start
End
Truncate
Loop
Slide (to move the sample left and right to get things lined up tempo/beat wise)
Delete section (to remove unwanted sounds from an area)

I’m been really into the idea of using auxy as more than a sketch pad. Pushing it to the limits and using it as one of my main DAWs
