TheRealJFalc January 16, 2019, 3:52am #1611
Yaad:

it would be awesome if pitching samples up or down didnt stretch them, and instead just changed the pitch

This is something that’s very tricky and actually tends to introduce artifacts into the sample. I don’t think it’s going to be implemented. However, you can do this in Audacity, which is a free desktop application. Then you can bounce the sample out into Auxy in the key you need it to be. Be warned there will be artifacts, but they might not be totally noticable if it’s only a semitone or so off.

NotMiles July 8, 2017, 6:05am #1615

I’ve made part of one of my songs in 5/4 so it’s doable. Just make a loop that’s 8 bars long, but then only make the scene 5 measures long (This gets you four measures with five beats in each measure). This will get you 5/4 time signature. 3/4 can be achieved by using the same method OR using triplets.

Here’s the song I made with a 5/4 section in it (skip to 1:58)
https://soundcloud.com/not_miles/umbra

As for 7/8, I haven’t quite been able to figure that one out, but it may be possible. I agree with you that a new feature in Auxy for this time signature would be helpful as well as for 5/8 and 9/8 stuff.

Hope this helps!
