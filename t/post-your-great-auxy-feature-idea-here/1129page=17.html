Xiejra:

around 50 instrument slots

That’s not possible with the way MIDI channels are designed.

akabillposters January 28, 2018, 7:28pm #344

Sure it is.
You think Ableton Live, Logic Pro, etc… can’t do 50 MIDI tracks?

I think you’re referring to the 8-channels limit or the 16-per-port factor. That’s basically a legacy, 5-pin cable limitation.
Modern software and processors and USB can handle many, many times that number of channels.

USB 2 (which is still the standard for MIDI USB [because there’s no practical need to go higher]) can support at least 8 ports, each containing 16 MIDI channels. That’s 128 MIDI channels possible through a basic USB 2 cable. Running internally on an iPhone, the number is likely to be closer to what a desktop DAW can work with.

Even using a modern sub-£75 MIDI hardware box, you’ll likely get 64 discrete channels, but then, who really has 64 hardware devices hooked up at the same time (let alone 128)?

 
Fwiw…
