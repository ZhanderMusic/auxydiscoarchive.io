Tarekith:

There’s no way to try different drum kits on a recorded sequence now? We have to create a new instrument and then copy the pattern data each time?

Yes, this is because the kit model doesn’t really apply to custom kits where you don’t know which type of drum is on which slot. We did a lot of thinking about this and decided to go for the kit approach we have now where you focus on the individual samples instead of kits. It feels like that’s the approach most good producers use, even though kits make sense in some contexts. For instance if you have an app where you can play “traditional drum kits”.
