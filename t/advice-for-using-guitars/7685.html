Advice for Using Guitars

Production Talk
DHO February 16, 2018, 9:30pm #1

I’ve been playing around with the new guitars and came across something that I think helps add an extra level of realness to these sounds. Since on guitars you often end up plucking strings at different volumes, I slightly automated the volume for each note. It takes a bit of time but makes a nice difference. Not sure if this is something the instruments already do automatically but I still found it helps.
