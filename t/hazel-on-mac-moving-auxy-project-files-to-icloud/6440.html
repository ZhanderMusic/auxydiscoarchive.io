Hazel on Mac - Moving Auxy project files to iCloud

Production Talk
MisterMaster January 20, 2018, 5:40am #1

I absolutely love the Mac application Hazel and have mentioned it once before in this forum. It lets you create rules to take actions when new files appear (and much more).

My latest rule is pretty simple, but really handy:

  • IF: An Auxy project file appears in my downloads folder.
  • THEN: Move it to my Auxy project folder, on iCloud.
  • AND: Rename the file to a name I chose, rather than using the file name.

This means:

  1. I can click any Auxy project link on my computer, choose the download option and in a few seconds it’s accessible via Auxy on my phone.

  2. Because I’ve asked Hazel to automatically MOVE the file to icloud (not copy), there’s no files left cluttering up my Mac.

It’s a paid app, but worth it. I totally adore apps that make stuff easy like this!

Oh yeah, my Mac also speaks “Moving Auxy Project to iCloud”, because, why not.

Here’s some of my rules, from an earlier thread: -

New desktop icons on my computer

If a new icon is dropped onto my desktop, it automatically files it away cleanly using the rules I defined. Images move to the pictures folder. Applications move to the Apps folder. And if an icon isn’t used for a while, it’s moved into an “Old” folder, for me to decide if I need it.

It also pulls any images that are messaged to me from iMessage and puts them in my Pictures folder.

And it sorts my downloads folder into a number of sub-folders, auto-magicly.

I made a temp folder on my desktop, where any files are deleted if not used for 2 days. Perfect for files needed for a few minutes only.

And finally - and this one’s quite geeky - If I speak to my Google Home, the IFTTT application saves a file in my dropbox cloud with the spoken command as the filename. That file syncs back to my Mac automatically via Dropbox and when Hazel sees the file appear locally, it runs an Applescript based on the name of the file. LOL. So, now I can speak to the Google Home and make things happen on my Mac.
