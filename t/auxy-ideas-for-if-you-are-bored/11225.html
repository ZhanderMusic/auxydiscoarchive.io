Auxy Prompts (2).jpg1588×2246 1.47 MB

So how it works it that if you are bored, write the numbers 1 - 100 on paper, cut them out and pick one randomly (or in the more modern age, use an online random number picker) and then create whatever the list tells you to create.

I know some are very similar but it was hard coming up with 100 ideas.

I’m very sorry for making the image version have a title that sounds like it came from a cheesy kids’ TV show.

Feel free to post links to any tracks you have created using this list as inspiration.

Ask me any questions you have about it.

Enjoy!
