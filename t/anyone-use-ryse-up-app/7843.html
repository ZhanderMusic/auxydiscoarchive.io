Anyone here use the RYSE UP app?

It’s a real mixed bag. Some clearly serious users with experience. But others try-hards make it feel like a hook-up app.

It does seem to have a heavy slant towards rap and r&b.

 
I know only a few here collab outside of the community, so I’m wondering if anyone here’s checked it out — and found it useful.
