Here are some packs I would love to see:

• Trap drum kits - I need some more trap drums. Future Bass is in the category of Trap I think, so Future Bass drums as well. Maybe also some hard trap stuff…

• Anthem Progressive House leads - I have been trying so hard to search for good kick drum samples for Anthem Progressive House and huge Anthem leads. This would be amazing.

• Electro House - Maybe add some snares good for EDM. I say buildup snares and FX snares. Maybe some Electro Kick Drums as well.

• Guitar - Possibly different guitar sounds and the ability to record our own tracks right into Auxy. That would be incredible.

• Drums - Ability to import our own samples and/or use a sampler so we can make buildups and tune drums.

• Bass - Add some more basses like Punch in the Original section. Maybe more 808 type basses for Trap.

• FX - Maybe some White Noise, Filter, Ducking Effect (so we can control when we want the instrument to sidechain the kick) (also the ability to sidechain drums other than the kick), Gate, Visual EQ, Compressor (Visual Display).

• Future Bounce - Probably introduce some new drum kits and instruments in this one.

• Jungle - Maybe add tom drums and jungle stuff

• Dubstep - Add some Wobble Basses and Growls. Also new Drum Kits

• Orchestral - Add some instruments for making an Orchestra or something

• Video Game - Add some video game style leads and some crunchy drums.

• Basic - Add some basic sound waves like sine, square, pulse, saw, etc.

That’s it. Hope that’s enough Just throwing out my ideas!
