Pinewoodog March 11, 2018, 4:47pm #15

I also have a Mac, and I have used both LPX and Ableton Live 9 Standard. When I was looking for a good Mac software I did a lot of research, so I’m pretty familiar with the top DAWs and top plugins.

I put together a list of the pros and cons of the top 2 DAWs for Mac; Logic Pro X and Ableton Live.

Logic Pro X:
Pros:
Easy to learn
Plenty of features
Amazing in-built synths
Tons of samples
Worth your money

Cons:
Difficult MIDI programming without a keyboard
Mac Only
64 bit plugin support only
Tough and confusing Automation

Ableton Live:
Pros:
Great workflow
Easy drum programming with “drum rack”
Amazing in-built effects
32 and 64 bit plugin support
Session View for LPA and smoother creation
Super easy Automation

Cons:
Steep Learning Curve
More Expensive
Doesn’t come with “good” in-built synths
