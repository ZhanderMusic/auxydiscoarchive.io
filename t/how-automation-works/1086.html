How automation works

Auxy Feedback
Auxy July 26, 2017, 8:38am #1

Please note: The Sweeper was unfortunately broken in 4.0 but it will be fixed if you update to 4.0.1.

Automation refers to automating sound settings over time. A simple example is to automate the volume to fade in an instrument, but that’s just scratching the surface of what you can do. Here’s what you need to know to get started:

Automation loops for each instrument

  • The big news is that we have added automation loops—a new type of loop that lets you automate all the sound settings.
  • The automation loops are added right below the melody loops for each instrument.
  • The transition feature has been removed and all projects that use transitions have been updated to use automation loops instead.
  • Just like melody loops, you select the automation loops that you want to be active in each scene.
  • Each scene can have one melody loop and one automation loop active for each instrument.


Automation loops below the melody loops.

Creating and editing automation loops

  • Select which sound setting you want to automate in the top left corner.
  • Tap the grid to add ‘nodes’ and build your automation curve.
  • Drag nodes right to increase the value and left to decrease.
  • Drag the curve to move a segment between two nodes.
  • Select and edit multiple nodes by long-pressing the grid and use the selection box.
  • Copy nodes by selecting them and then long-pressing the grid to paste.


Volume automation with a couple of nodes.

Automation length

  • Note that each automated setting has its own length, which means you can make a loop that repeats e.g. a filter change every bar, while the volume is repeating every 4 bars.
  • You can set automation length to ‘Scene’, which will make the curve as long as the current scene.

Snapping to previous and next scenes

  • If you remove the start- and end nodes (marked with a dot inside the circle), the curve will automatically snap to the values of the previous and next scenes.
  • This is useful for making smooth transitions between scenes.


Removing the start node snaps the curve to the value of the previous scene.

Setting static values

  • Sound settings are no longer stored with each scene.
  • If you change e.g. the volume of an instrument in the tweak panel, it will change the volume across all scenes where volume is not automated.
  • If you want specific values in a scene, you have to use an automation loop and set ‘static’ values, i.e. straight automation curves.
  • The easiest way to set static values is to tap the grid once to add a straight line, and then drag the curve left or right.

New sound tweaks

  • Each sound now has a pan setting, which lets you pan the sound to the left or right.
  • Each sound now has a highpass setting, which lets you filter out low frequencies to improve the mix or create sweeping effects.
  • If a sound is missing a certain setting, it will be grayed out in the tweak panel and automation editor.


The new tweak panel with pan and highpass.

Pitch automation

  • Selecting ‘Pitch’ lets you pitch each instrument one octave up or down.
  • Nodes will automatically snap to even semitones.

Changes to how the Sweeper works

  • The ‘Sweeper’ effect no longer has start and end values.
  • To create a basic sweep, draw the notes and add automation to the lowpass setting.

Feeling confused?

Post your questions in the comments below and we’ll do our best to explain.

Hope you like it!
