akabillposters January 3, 2018, 5:59pm #99

Can you elaborate?

Me?

That first track preview was a strange choice.

Opening with pianos… rather than something that screams ‘new sounds’ straight out of gate – especially when new sounds are what everyone’s waiting for.
I’m hoping/assuming they haven’t used time and resources adding more pianos when there’s already an entire piano pack (and those pianos in the preview sounded like the existing pack).

The early snares sounded like up-pitched Oregon/Pacific, but the later toms sounded great. The kits sounded more interesting in the later parts, so some decent new kits coming.
(Some acoustic drums [samples] would have been great.)

Vocal chops presets are possibly a risky option. If there’s too few manipulation options, then they’re going to get creatively exhausted quite quickly.

 
Overall, to me, it sounded a bit like it’s doubling-down on existing genres, rather than adding anything that’s going to broaden the options/appeal.

Hopefully there are more high-points that weren’t in the preview, but time will tell.

I think this update might be more about features and workflow changes for me, as the AP previews suggest I won’t be getting much new value there.

 
(Sorry for being a less-than-positive voice.)
