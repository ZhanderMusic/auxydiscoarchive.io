NickElle September 7, 2018, 1:43am #30

Is Aum a DAW though?
It seems to be just a mixer, but I’ve never used it, so I may be wrong.
A DAW should be able to do everything.
It should be able to record, sequence, mix, etc.
It should be the end all of gear. Everything should run into the DAW, and the DAW should be able to capture the data being thrown at it, and you should be able to freely do things with that data, putting it in place, editing it, processing it.
That’s what I consider a DAW.
And, I consider a “pro DAW” to be a piece of software that can do all those things at a very in-depth level.
