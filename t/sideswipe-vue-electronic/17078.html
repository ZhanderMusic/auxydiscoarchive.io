Saturn Vue

The Saturn Vue is a compact SUV that was sold and built by Saturn, and it was Saturn's best-selling model. It was the first vehicle to use the GM Theta platform when it was introduced in 2001 for the 2002 model year. The Vue was later facelifted for the 2006 model year. A second generation model was launched in 2007 for the 2008 model year as a rebadged Opel Antara.
