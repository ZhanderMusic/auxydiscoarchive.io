HYDRANS June 20, 2019, 12:57am #11

There isn’t one specific way I process a sample. For me, it really depends on the specific sample that I am processing and the song that it is going to be used in

For example, if I am processing a vocal sample for something more mainstream/melodic sounding, I might use simple techniques such as slight EQing and some reverb for subtle effects. I wouldn’t want to take focus away from the actual vocal performance. However, if I was using the same vocal for a more heavy, experimental track (what I normally make) I would probably use various effects to fit the song. I like to use OTT, distortion and a subtle amp effect along with EQ to get a blown out speakers effect to a vocal sample.

TL;DR

It really depends on the sample and type of song I’m using it for
