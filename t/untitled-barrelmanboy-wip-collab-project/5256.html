akabillposters December 20, 2017, 2:21pm #6
  • Tap on the arrow on the right edge of the screen to bring out the sidebar
  • Bottom of that: ‘Export’
  • Export window: Tap ‘Export’ (bottom-right)
  • Tap on ‘…More’
  • ‘Export Individual Instruments’

This basically exports each instrument as a separate track.

Once it’s done mixing/mastering all the tracks, it’ll want to save a ZIP file containing all the WAV files.
You’ll need to save them somewhere – e.g Dropbox, Apple Files, etc…

Save it, then share the link to that file.
