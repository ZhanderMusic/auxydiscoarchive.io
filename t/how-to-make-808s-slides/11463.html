when you want to try to attempt a slide, on the actual piano roll, raise the note by one octave and right before that note, automate the pitch of the 808 up so it “slides” into it and sounds more natural and when the high octave note hits get rid of all pitch automation so it sounds like it’s balanced out with the pitch up you did right before it… i could get you a project file tomorrow that shows an example of that, it’s one in the morning here so sorry if that didn’t really make sense
