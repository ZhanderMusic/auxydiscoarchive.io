hello everyone, new to this forum and not yet an expert in auxy…

question: is there a way to delete an automation? let say i made an automation for pitch, for volume, for lowpass, etc. and then later on decided to remove just one of those automation. how do you do that - to remove one automation but keep the other automations as is?

akabillposters September 14, 2017, 9:51pm #2

You need to tap on the dots/points you’ve created to remove each one, until none are left.
This effectively resets the automation to 0/none, in which case, what level you’ve set in the main Instrument controls panel will be used.

Hope that helps.

NEWNAMES September 14, 2017, 9:51pm #3
Ideas for improving Auxy

You can delete the individual nodes like induvidual notes. Just select the ones you want to delete, and then just tap on one node to delete all of the selected nodes. Hope that helps!
