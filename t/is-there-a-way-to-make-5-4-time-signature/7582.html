akabillposters February 14, 2018, 8:45am #9

That, on its own, is not what a 5/4 time sig means.
A 5/4 time sig would be like a simple 4/4, but with an extra beat per bar.

What you’re suggesting is an extra bar per pattern.

To clarify, to emulate a 5/4 signature in Auxy, you will need to use 8-bar patterns, using only the first 5 bars, and then set section length to 5 in the sidebar.

Unfortunately, this ‘faking it’ approach means your 5-bar (8) pattern will go ‘out of sync’ with the bars UI in the app, as you’ll effectively be delaying the start of the next bar by one beat every time. This causes a knock-on effect that takes 5 bars in Auxy to resolve cleanly.

So, it’s doable, but it’s messy (in Auxy).

https://app.auxy.co/projects/rsoYkPUGNAENCLmwHaoT1Q==

 
If you wanted to do a 5/4 signature, in Auxy, with repeating segments that don’t use 4 bars (actual bars, not Auxy UI bars), then you’re in for a world of pain as you have to use a really horrible patchwork of bars and loop lengths to get it done.

Personally, if you’re attempting the above, I’d use something other than Auxy. It isn’t worth the pain.
