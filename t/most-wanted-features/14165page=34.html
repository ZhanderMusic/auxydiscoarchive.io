ThoughtfulNoise April 26, 2021, 4:14pm #688

Automations can be set for 1, 2, 4, 8 bars or Scene.
Currently, there are only 16 grid snap points across the whole Scene. 1/16, 1/32, 1/12, 1/24 aren’t available.

1, 2, 4 and 8 bar automations all support Free mode, but not Scene, which only supports 17 vertical axes (16 cols + 0) across the entire Scene.

This means automations across a Scene are very limited by comparison.

image1204×900 36.2 KB

(Note: It’s possible to select ‘Free’ in 1, 2, 4, 8 bar automation canvasses, but it reverts to 17 vertical grid axes when switching to Scene, even though it still says ‘Free’. If you’d previously selected 1/16, 1/32, etc… and switch to Scene, the 1/16th, etc… is greyed out.)

If you set up Free automation points in 1, 2, 4, 8 bar canvases and then switch to Scene, points that don’t fall on the 16-column grid are discarded.)
