averageteencuber May 30, 2020, 8:11am #378

a few other small quality-of-life type features I’d like to see:

  • other modes added to go along with major and minor
  • changing all the selected notes to soft with one tap
  • adding the feature where you tap on either side of the tempo slider to move it up or down one to all of the new Auxy 6 automation sliders
  • a wider range of tempos, maybe like 30–1000 bpm
  • the option to keep an imported sample in a project when sharing with others, so that they can download it and use it for their own tracks
  • the ability to edit the ducker/sidechain curve to get the best possible sounding sidechain
  • adding key changes at the tap of a button or something
  • adding the “tune” feature (from drum kit samples) to the melodic instruments, as well as increasing its range to ±100 cents. basically detuning
  • automation for the compress filter on drum kits

I may edit this if I or others come up with more features. this list is only some of the small things anyway, I have quite a few bigger ideas for features as well (such as different time signatures, but I know that’s been mentioned before).

1: Instrument Mute and Solo in the main/instruments/loops view

Swipe from the left edge to reveal (push-to-right) a column with Mute and Solo controls, but with similar functionality as desktop DAWs — i.e. can ‘solo’ single or multiple instruments (maybe c/o a modifier key* or long press to enable solo for multiple instruments/tracks.

Rendering/exporting would simply ignore muted instruments/tracks.

 

2: A modifier button

A modifier key would effectively double the possible actions.
Plenty of options, e.g.…

• Shift + hold on note could bring up a vertical velocity slider
• Shift + tap on sequence of patterns in the track construction bar would allow us to select a sequence of scenes, rather than just single scenes, which becomes really handy if you upgrade the Loop function as mentioned in my previous post.

 

3: Export labels to ALS…
• Export Scene Labels (Auxy track construction bar) as Locator Marker labels (for Arrangement View) when exporting to Ableton Live Set**
• Export Loop/Pattern labels, for both midi and audio stems?

They’re useful and good composition practice.
