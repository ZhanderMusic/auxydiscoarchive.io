TC64 April 16, 2021, 5:53pm #656

You use an app on PC or Mac like Polyphone, where you set up sample key zones, loop points, and ADSR in a similar way to how instrument creation in Logic’s EXS24 works. Or, download a pre-existing .SF2 from the web, as there’s already many out there.

.SF2 files can contain multiple instruments (127 per “bank”, like MIDI), so if they were imported into Auxy, you’d choose which bank and instrument in the .SF2 you’d want to use.

If SoundFont support is considered, I’d suggest implementing .SF2 first before .SFZ, as most SoundFonts available online are not in .SFZ format—you’d have to convert them, which is inconvenient.
