Auxy February 23, 2021, 12:09pm #505
ThoughtfulNoise:

Finish out the A DS R, or at least Decay. Automating decay is a very popular effect, in the targeted genres and beyond

This is not as straightforward as it sounds, since sounds are not simply waveforms fed through an amp envelope. However, we’re looking at retrofitting a shape macro to a lot of sounds where it would make sense to have it. E.g. leads but not pads. Then we would set a low sustain as default and max decay (which is 5 secs in our engine). Turning down shape would then shorten decay to allow for the progressive “shortening” of sounds that I assume you’re looking for?

Auxy February 23, 2021, 12:15pm #508
ThoughtfulNoise:

Add support for importing sound packs (created using aforementioned synth tool)
(This addresses the requests for an integrated synth UI directly in the app via a much more practical/achievable route. It could open up new additional revenue stream for Auxy, but also creators, if you set up an official marketplace through which user-created sound packs can be sold/downloaded.
Additionally, the ability to create and use your own and 3rd-party sound packs could be a significant differentiator for Auxy amongst its direct competitors. And not least, this new import function could easily justify a new higher Subscription tier.)

We would love if this was doable. What we’ve learned, though, is that for this to work, you need to give the users and the sound designers access to the same tool. I.e. like any plugin where you can buy packs that have been created with the same synth that you have access to. The best approach for us would probably be to build a dedicated synth app that you can make sounds with and export to Auxy. Like I mentioned, these would probably be pretty lame in comparison to what you get in the packs, regardless.
