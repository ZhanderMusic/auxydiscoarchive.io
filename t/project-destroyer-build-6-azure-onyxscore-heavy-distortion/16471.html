AzureOnyx August 17, 2019, 12:48am #3

To put it simply, the problem with this update that I personally experienced, and with others, is from the inability to replicate Auxy 5 when using the new automation tweaks.

I use strings pretty often because the style of trance that i’m producing does heavily use strings. At the moment, using the strings collection is a lot harder to use than how it was. I made a post awhile ago explaining this problem, but basically the strings don’t merge together as seamlessly as A5, you can audibly hear the attack in A6 now. Pads like silk was perfect for replicating strings and I do use it from time to time if I want to layer the long chords. Now it’s used for genres like future bass where silk was heavily used.

I did lightly use automation in A5, but I can’t use it as fluently as before, knowing that the automation nodes, or dots, don’t snap into equal value, with the exception of pitch. Right now, I can only do simple automation, reverting me back to when Auxy used “Transitions” rather than A5’s automation.
